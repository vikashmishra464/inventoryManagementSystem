## Quick Installation

    git clone https://github.com/vikashmishra464/inventoryManagementSystem.git

    cd Inventory_Management_System
    
### Composer

    composer update
    
 
    
 ### For Migration table in database [Create database name as ```IMS```]
 
    php artisan migrate
    
### Server ON ```url: http://127.0.0.1:8000/```

    php artisan serve
